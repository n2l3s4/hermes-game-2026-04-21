# Bobo's Hide and Seek Safari

A fun animal finding game for kids! 🦁🐵🐘

## Description

Help Bobo find all the hidden animals in the safari! Cute animals are peeking out from behind the bushes and trees — click on their eyes to find them. Can you find all 8 friends?

## How to Play

1. Look for peeking eyes hiding in the jungle scene
2. Click or tap on the eyes to reveal the animal
3. Find all 8 animals to win!
4. Listen for fun sound effects when you find each friend
